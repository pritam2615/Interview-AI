📌 Interview-AI

An intelligent AI-powered interview preparation platform that helps users practice, analyze, and improve their interview performance through automated question generation and feedback.


🚀 Features

🎯 AI-Generated Interview Questions
Generate role-based technical and behavioral questions.
🧠 Smart Answer Analysis
Analyze user responses using AI and provide constructive feedback.
📊 Performance Evaluation
Get insights into strengths, weaknesses, and improvement areas.
💬 Mock Interview Simulation
Simulate real interview scenarios for better preparation.
⚡ Real-Time Assistance
Fast and responsive system for seamless interaction.


🛠️ Tech Stack

Frontend:
React + Vite 

Backend:
Node.js + Express,
MongoDB + Mongoose,
JWT Auth,
CORS, dotenv

AI Integration:
Gemini API
